## Analysis

### 1. Wat is de "local baseline"?

De lokale concurrenten in de Betuwe/Rivierenland opereren op een **opvallend laag niveau**:

- **Design**: Verouderde templates (Drupal, standaard WordPress), geen moderne visuele identiteit. HMS Schoonmaak en Van Haaften zien eruit als 2015-era websites.
- **Trust signals**: Vrijwel afwezig. Geen certificeringsbadges op homepages, geen of minimale Google Reviews integratie, sporadische testimonials zonder scores.
- **CTA's**: Zwak en generiek. "Offerte aanvragen" verstopt in navigatie of op aparte pagina. Geen sticky CTA's, geen multi-step formulieren, geen urgentie-microcopy.
- **Content**: Dunne dienstbeschrijvingen, geen prijsindicaties, geen processtappen, geen FAQ's.
- **Positieve uitzondering**: Clean Up Tiel steekt er bovenuit met een 9.7/10 score, WhatsApp, persoonlijk aanspreekpunt, en klantenlogo's.

**Baseline samenvatting**: Een redelijk moderne website met zichtbare reviews, VCA-certificering, en een duidelijk contactformulier zou al in de top 2 van de regio staan.

### 2. Wat doen de nationale leiders anders?

De nationale leiders onderscheiden zich op vijf vlakken:

1. **Certificeringsstapeling**: Hensen (6 certificeringen), Balans (6 certificeringen), CSU (meerdere awards). Lokale bedrijven tonen er 0-1.
2. **Review-architectuur**: Balans toont 9.9/10 met benoemde bedrijfsreviews. Hensen heeft 10+ vijfsterren reviews. Lokaal: nul tot drie losse testimonials.
3. **Transparantie-innovatie**: Hensen biedt een online portaal waar klanten uren, frequenties en prijzen kunnen monitoren. Balans heeft klant- en medewerkersportals. Dit niveau van transparantie bestaat lokaal niet.
4. **Mensgerichte branding**: Eigenaren/directeuren zijn zichtbaar (Hensen's Joost Kleinjan op LinkedIn, Nederrijn's drie directeuren bij naam). Lokaal is alleen Clean Up Tiel's "Maxim" persoonlijk.
5. **Content-diepte**: Sectorspecifieke landingspagina's (onderwijs, zorg, kantoren, leisure), gedetailleerde werkwijze-pagina's, duurzaamheidsrapporten. Lokaal: één generieke dienstenpagina.

### 3. Welke fouten maken de slechtste concurrenten?

1. **Geen trust signals**: Van Haaften claimt 45 jaar ervaring maar toont nul reviews, nul certificeringen, nul klantenlogo's. Puur zelfverklaarde geloofwaardigheid.
2. **Onzichtbare contactmogelijkheden**: HMS Schoonmaak heeft geen telefoon prominent, geen formulier op de homepage, geen WhatsApp. De bezoeker moet zelf op zoek naar contactinformatie.
3. **Verouderd design als schoonmaakbedrijf**: Een gedateerde website is voor élk bedrijf slecht, maar voor een schoonmaakbedrijf is het ironisch en direct schadelijk — het ondermijnt de kernbelofte.
4. **Geen scheiding B2B/B2C**: Bedrijven die beide markten bedienen mengen zakelijke en particuliere informatie op dezelfde pagina zonder duidelijke paden.
5. **Homepage zonder inhoud**: Meerdere sites (inclusief John's huidige site) hebben homepages met minimale of placeholder-content.
6. **Recruitment boven klantconversie**: Nederrijn (#1 van NL) prioriteert "Solliciteer direct" boven klantenwerving op de homepage — een strategische keuze, maar vanuit conversie-perspectief een gemiste kans.
7. **"Meer info" als CTA**: Oceaan gebruikt "Meer Info" als primaire CTA — generiek, geen actie-urgentie, geen waardepropositie in de knoptekst.

## Lowest-Rated: What Makes Them Look Bad

### HMS Schoonmaak (5/10)

1. **Divi template syndrome**: Generic WordPress/Divi theme (v4.27.6) with zero custom branding. Screams "template."
2. **Bloated CSS**: ~50KB+ unminified inline CSS with unused animation keyframes, redundant breakpoints.
3. **Typography failure**: Generic Open Sans, no weight variation, 14px body text feels cramped, uniform 1.7em line-height with no heading/body differentiation.
4. **Single-color dependency**: Only accent color is `#2ea3f2` (blue). Body text `#666` has poor contrast. No secondary/accent hierarchy.
5. **No hero section**: No visual hook above the fold. Text-only "welkom" messaging.
6. **Restrictive container**: Fixed 80% width, max 1080px — feels narrow on modern screens.
7. **Zero trust signals**: No reviews, no certifications, no logos, no testimonials. Nothing.
8. **Outdated interaction**: Mobile menu uses select dropdowns (2012 pattern).

### Oceaan Schoonmaakservice (5/10)

1. **Avada template fatigue**: WordPress/Avada with heavy jQuery plugins. Feels 2015-era.
2. **Branding disconnect**: "Oceaan" concept is solid but execution doesn't deliver — muted green (#5db855) instead of ocean blues/teals. Inconsistent secondary colors (#5bdccd, #1cd6c3).
3. **Visual hierarchy chaos**: Multiple competing font sizes (12.75px, 20px, 25.5px) without rhythm. Tight line-heights.
4. **Generic CTAs**: "Meer Info" buttons everywhere — zero value proposition in button text.
5. **Inconsistent spacing**: Some sections have excessive whitespace, others are cramped.
6. **No modern UI patterns**: Basic buttons, no microinteractions, static design.
7. **Weak trust implementation**: Claims "gecertificeerd" but shows no certification badges.

**Common thread**: Both sites are WordPress premium themes (Divi, Avada) used out-of-the-box with minimal customization. The template DNA is immediately recognizable and signals low investment.

## Design Decisions to Incorporate (Top 5)

### 1. Single font family with weight-based hierarchy (from CSU)
One professional sans-serif (e.g., Inter or a similar clean font) used at weights 400/500/600/700. No second display font. Typography differentiation comes from size + weight + color, not font swapping. This creates quiet professionalism that template sites can't match.

### 2. Strategic full-bleed rhythm breaking (from CSU + Hensen)
Alternate between contained max-width sections (~1200-1400px) and full-bleed sections (hero, testimonials, CTA banner). CSU does this with the sustainability section (image-left, overlapping text box). Hensen does it with video backgrounds. This prevents the "stacked boxes" monotony that plagues template sites. Target: 2-3 full-bleed moments per homepage.

### 3. Certification badge repetition strategy (from Hensen)
Display VCA + any other certifications in three locations: (1) hero area / above the fold, (2) mid-page trust section, (3) footer. Hensen shows 6 badges in all three positions. For John's, even just VCA displayed this way would outperform every local competitor who shows zero.

### 4. Pill-shaped CTAs with restrained color (from CSU + EW)
Fully rounded buttons (`border-radius: 9999px` or `4rem`). Single accent color for primary CTA, dark neutral for secondary. Hover states with subtle shadow elevation (`0 4px 4px rgba(0,0,0,.1)` → `0 8px 8px rgba(0,0,0,.15)`). This detail alone separates modern sites from template sites using default square/slightly-rounded buttons.

### 5. Named social proof with visual anchors (from Hensen + Clean Up Tiel)
Testimonials from named people at named companies, displayed as cards with star ratings and company logos. Not anonymous "Klant uit Tiel" quotes. Hensen shows Exact, Stena Line with stars; Clean Up Tiel shows DBI Techniek with 9.7/10 score. For John's: collect 3-5 named client reviews with company names for maximum impact.

## CSS Layout Patterns That Create Professional Quality

### 1. Asymmetric grid splits
```
/* CSU sustainability pattern: 60/40 with overlap */
grid-template-columns: 1.5fr 1fr;
/* Image spans full height, text box overlaps with negative margin or grid overlap */
```
Breaks the monotony of equal-width columns. Used by CSU for image+text sections and Hensen for CEO narrative.

### 2. Contained-to-full-bleed section alternation
```
/* Contained section */
.section-contained { max-width: 1400px; margin-inline: auto; padding-inline: 24px; }
/* Full-bleed section */
.section-bleed { width: 100vw; margin-left: calc(-50vw + 50%); }
```
CSU alternates ~3 times per page. This rhythm creates visual interest without relying on decorative elements.

### 3. Card elevation system
```
/* Resting state */
box-shadow: 0 4px 4px rgba(0,0,0,.1);
/* Hover state */
box-shadow: 0 8px 8px rgba(0,0,0,.15);
transition: box-shadow 0.2s ease, transform 0.2s ease;
transform: translateY(-2px);
```
Every professional site uses this pattern. Both HMS and Oceaan lack it entirely, which makes their cards feel flat and static.

### 4. Section padding variation
```
/* Hero: generous */
padding-block: 120px 80px;
/* Content sections: moderate */
padding-block: 64px;
/* CTA banner: compact but impactful */
padding-block: 48px;
/* Trust strip: tight */
padding-block: 32px;
```
Varying padding creates visual rhythm. Template sites use identical padding everywhere, creating monotony. CSU uses 40-60px, Hensen goes to 60-100px, creating a "breathing" page.

### 5. Logo grid with consistent sizing
```
/* Client logos: uniform height, flexible width */
.logo-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(120px, 1fr)); align-items: center; gap: 32px; }
.logo-grid img { height: 48px; width: auto; object-fit: contain; filter: grayscale(1); opacity: 0.7; }
.logo-grid img:hover { filter: grayscale(0); opacity: 1; }
```
EW shows 12+ logos this way. Grayscale-to-color on hover is a subtle but powerful pattern that all three top competitors use for client logos.
