## Welke gegevens verzamelen we?

### Via het contactformulier

```
Als je het contactformulier invult, vragen we om:

- Je voornaam en achternaam
- Je bedrijfsnaam
- Je e-mailadres
- Je telefoonnummer (niet verplicht)
- De gewenste dienst
- Een toelichting op je aanvraag (niet verplicht)

We slaan deze gegevens op zodat we je aanvraag kunnen beantwoorden
en een offerte kunnen opstellen. Dat is alles. We gebruiken je
gegevens niet voor marketing of nieuwsbrieven, tenzij je daar zelf
om vraagt.
```

### Via de website (cookies en analytics)

```
We gebruiken cookies op onze website. Er zijn twee soorten:

Noodzakelijke cookies: deze zijn nodig om de website goed te
laten werken. Denk aan het onthouden van je cookievoorkeuren.
Deze plaatsen we altijd.

Analytische cookies: we gebruiken [Google Analytics / Plausible /
geen analytics — invullen] om te begrijpen hoe bezoekers de
website gebruiken. Welke pagina's worden bekeken, hoe lang
mensen blijven, via welke zoekterm ze binnenkomen. Deze cookies
plaatsen we alleen als je daar toestemming voor geeft via de
cookiemelding.
```

## Waarom verzamelen we je gegevens?

```
We verwerken je persoonsgegevens voor de volgende doelen:

- Je aanvraag beantwoorden en een offerte opstellen
- Contact met je opnemen over je aanvraag
- Onze website verbeteren op basis van gebruiksstatistieken

We sturen je geen reclame, nieuwsbrieven of aanbiedingen, tenzij
je daar uitdrukkelijk om vraagt.
```

### Rechtsgrond

```
- Contactformulier: gerechtvaardigd belang. Je neemt zelf contact
  met ons op en verwacht dat we reageren.
- Analytische cookies: toestemming. We plaatsen deze pas nadat je
  akkoord gaat via de cookiemelding.
```

## Met wie delen we je gegevens?

```
We delen je gegevens niet met derden. Niet voor marketing, niet
voor onderzoek, niet voor welk ander doel dan ook. Punt.

De enige uitzondering: als we wettelijk verplicht zijn om gegevens
te verstrekken (bijvoorbeeld op verzoek van de Belastingdienst of
politie). Maar dat is nog nooit voorgekomen en daar verwachten we
ook niet veel van.
```

## Hoe lang bewaren we je gegevens?

```
Gegevens uit het contactformulier bewaren we maximaal 12 maanden
na het laatste contact. Na die tijd verwijderen we je naam,
e-mailadres en telefoonnummer.

Factuurgegevens bewaren we 7 jaar, omdat de Belastingdienst dat
vereist.
```

## Jouw rechten

```
Je hebt de volgende rechten met betrekking tot je persoonsgegevens:

- Inzage: je mag opvragen welke gegevens we van je hebben.
- Correctie: kloppen je gegevens niet? Dan passen we het aan.
- Verwijdering: je kunt vragen om je gegevens te laten verwijderen.
- Bezwaar: je kunt bezwaar maken tegen de verwerking van je gegevens.
- Overdracht: je kunt vragen om je gegevens in een standaardformaat
  te ontvangen.

Stuur een e-mail naar info@johnsschoonmaakbedrijf.nl als je gebruik
wilt maken van een van deze rechten. We reageren binnen 30 dagen.
```

## Cookies

```
Bij je eerste bezoek aan onze website tonen we een cookiemelding.
Daarin kun je kiezen welke cookies je accepteert.

Noodzakelijke cookies kun je niet weigeren. Die zijn nodig om de
website te laten werken. Analytische cookies plaatsen we alleen
als je akkoord gaat.

Je kunt je keuze op elk moment wijzigen via de cookie-instellingen
onderaan de website.
```

## Beveiliging

```
We nemen passende maatregelen om je gegevens te beschermen. De
website gebruikt een SSL-verbinding (https), zodat gegevens
versleuteld worden verzonden. Formuliergegevens worden opgeslagen
in een beveiligde omgeving.
```
