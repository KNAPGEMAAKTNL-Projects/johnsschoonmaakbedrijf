# John's Schoonmaakbedrijf - Website

Professional website for John's Schoonmaakbedrijf, a commercial cleaning company serving businesses in the Betuwe region, Netherlands since 2007.

## Tech Stack

- **Framework:** Astro 5 (Static Site Generation)
- **Styling:** Tailwind CSS 4
- **Deployment:** Cloudflare Pages

## Development

```bash
cd website
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
